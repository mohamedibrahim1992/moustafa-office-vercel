# مكتب مصطفى حمزة — نظام إدارة المحاسبة

نظام محاسبة متكامل مبني بـ React + Node.js + PostgreSQL.

## التقنيات
- **Frontend:** React 19 + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** PostgreSQL (Supabase)
- **Hosting:** Vercel

## متغيرات البيئة المطلوبة على Vercel
| المتغير | الوصف |
|---|---|
| `DATABASE_URL` | Supabase connection string |
| `SESSION_SECRET` | كلمة سر عشوائية طويلة |
| `NODE_ENV` | `production` |
